Practice Repo

Ship tiny changes on trunk.

This repository is set up for short, verified steps. Keep each change small, run npm test, then commit directly to main or merge a branch immediately.

  1. Pick one small change that can be finished in a few minutes.
  2. Make the smallest edit that moves the code forward.
  3. Run the tests before every commit.
  4. Commit often with clear messages that describe one completed slice.
